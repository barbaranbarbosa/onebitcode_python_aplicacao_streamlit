# Projeto Streamlit
Aplicação desenvolvida em Streamlit para geração de tabelas, métricas e gráficos a partir de um banco de dados.

## Objetivo
Explorar as principais bibliotecas Python voltadas para manipulação e visualização de dados, com foco no uso do Streamlit.  
O projeto foi versionado no GitHub e integrado ao _streamlit.io_ para realização do deploy da aplicação.

## Descrição
A aplicação é composta pelos seguintes módulos:
- **Dataset**: exibe o conjunto de dados de vendas, com filtros disponíveis na barra lateral.
- **Receita**: apresenta gráficos e tabelas resumo com as principais análises.
- **Vendedores**: exibe métricas relacionadas ao desempenho dos vendedores, destacando o vendedor mais produtivo.

## Como usar
1. Clone o repositório ou baixe os arquivos do projeto.
2. Crie e ative um ambiente virtual.
3. Instale as dependências listadas no arquivo `requirements.txt`.
4. Execute a aplicação no terminal com o comando:
```bash
streamlit run app.py
